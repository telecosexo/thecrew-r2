# 🧊 Painel ICE 🧊 
Sistema web desenvolvido para gestão em FiveM (GTA RP), focado no controle de **Vendas**, **Cálculo de Produção**, **Lavagem de Dinheiro** e **Registro de Ações**.

## Funcionalidades

### Vendas & Produção
- **Catálogo Interativo:** Seleção de armas por categoria (Pistolas, Subs, Fuzis, etc.).
- **Tabela de Preços Dinâmica:** Alternância global rápida entre valores de **Parceria (Mínimo)** e **Pista (Máximo)**.
- **Carrinho de Compras:** Adição de múltiplos itens, com ajuste de quantidade e atualização automática de valores.
- **Calculadora de Materiais:** Gera automaticamente a lista de materiais necessários (Alumínio, Cobre, etc.) baseada nos itens do carrinho.
- **Logs no Discord:** Envia um relatório detalhado da venda (Vendedor, Facção, Lucro) via Webhook.

### Registro de Ações
- Registro de PVP/RP (Local, Resultado e Horário).
- Lista de participantes dinâmica.
- Envio de relatório formatado para o Discord.

### Utilitários
- **Divulgação:** Botão de cópia rápida para anúncios da loja.
- **Mascote:** Mascote interativo (Pixel Art) fixo na interface.
